Indonesia CSV Map Plotter Guide

A lightweight, browser-based tool for plotting coordinate data from CSV files onto an interactive map of Indonesia. This application runs entirely in the client's browser—no server setup required.

Features

  • Privacy-Focused: All CSV parsing and rendering happens locally in your browser.
  • Flexible CSV Support: Automatically detects common column names for latitude, longitude, and labels.
  • Multiple Map Styles: Standard, Satellite, Light/Dark, and Vector-only modes.
  • Export Capabilities: Export as PNG or SVG (Vector).

Usage Guide

1. Preparation

Prepare your data in a .csv file. The file must have headers.

Example CSV:

City,Lat,Lon,Description
Jakarta,-6.2088,106.8456,Capital City
Bali,-8.4095,115.1889,Tourism Hub
Surabaya,-7.2575,112.7521,Industrial Center

2. Plotting Data

  • Click Choose File in the control panel.
  • Select your .csv file.
  • If the map doesn't plot automatically, check the Column Mapping section.

3. Customizing the View

  • Use the Map Style dropdown to switch styles.
  • If using Indonesia Only (Vector), color pickers allow you to change the land, sea, and border colors.

4. Exporting

  • For PNG: Use "Select Area" to crop, select resolution, then "Download PNG".
  • For SVG: Select the "Indonesia Only (Vector)" style, then click "Download SVG".

Source code available at: github.com/ftharyanto/plotter
